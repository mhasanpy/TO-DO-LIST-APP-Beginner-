A simple yet powerful **command-line To-Do List Application** built with Python. This is **Project 1** in my series of 40 Python exercises, focusing on fundamental programming concepts.

## 🎯 Purpose

This project demonstrates practical implementation of core Python concepts by building an interactive task manager. Users can create, read, update, delete, and persist tasks using file storage.

## ✨ Features

- ✅ Add new tasks with titles and descriptions
- 📋 View all tasks with their status (Pending/Completed)
- ✔️ Mark tasks as complete
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- 💾 Automatic saving to a local file (`tasks.txt`)
- 🔄 Load tasks automatically when the app starts

## 🧠 Concepts Covered

This exercise practices the following Python fundamentals:

| Concept | Implementation |
|---------|----------------|
| **Lists** | Storing multiple tasks in memory |
| **Dictionaries** | Representing individual tasks (id, title, description, status) |
| **File Handling** | Reading/writing tasks to `tasks.txt` for persistence |
| **CRUD Operations** | Create, Read, Update, Delete functionality |
| **Loops & Conditionals** | Main program loop and menu choices |
| **Functions** | Modular design with separate functions for each operation |
